# WC Coupon Enhanced

WooCommerce 折價券強化外掛，提供指定物流免運、My Account 個人化優惠券頁面。

**Version:** 1.4.0
**Author:** jimmy-is-me
**License:** GPL2
**Requires:** WordPress 5.8+, WooCommerce 6.0+, PHP 7.4+

---

## 功能說明

### 一、指定物流免運

在折價券後台新增「指定免運」Tab，可針對單一折價券設定指定的物流方式直接免運費。

- 啟用開關：預設關閉，開啟後才生效
- 勾選的物流方式 → 套用此折價券時直接免運費
- 未勾選的物流方式 → 維持原本收費規則，不受影響
- 支援多個運送區域與多種物流方式同時勾選

### 二、My Account 我的優惠券

在會員中心新增「我的優惠券」頁面（`/my-account/my-coupons/`）。

- 只顯示該登入使用者**有資格使用**的優惠券
- 若折價券設有 Email 限制，只有對應 Email 的使用者才能看到
- 沒有 Email 限制的折價券，所有登入會員皆可看到
- 顯示每張優惠券的使用條件：折扣內容、最低/最高消費、每人限用次數、是否可合併等
- 自動過濾：已過期、已達整體使用上限、個人已達使用上限的折價券不顯示

## 安裝方式

1. 下載本 repo 的 ZIP 或 clone 至 `/wp-content/plugins/wc-coupon-enhanced/`
2. 至 WordPress 後台 **外掛 → 已安裝的外掛** 啟用「WC Coupon Enhanced」
3. 啟用後系統會自動刷新固定連結（Rewrite Rules）
4. 若「我的優惠券」連結出現 404，請至 **設定 → 固定連結** 按一次「儲存設定」

## 使用方式

### 指定免運設定

1. 前往 **WooCommerce → 折價券** 編輯或新增一張折價券
2. 點選上方 Tab 列的「指定免運」
3. 勾選「啟用『指定免運』功能」
4. 在列出的物流方式中勾選要免運的物流
5. 儲存折價券

### Email 限制設定（個人專屬優惠券）

1. 編輯折價券，進入「使用限制」Tab
2. 在「允許的電子郵件」欄位填入指定 Email
3. 儲存後，只有該 Email 的登入會員可在「我的優惠券」頁面看到此券
